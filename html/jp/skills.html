スキル

プログラミング言語

JAVA

JAVAは、狭義ではプログラミング言語JAVAを指す。

C#

C#は、アンダース・ヘルスバーグが設計したプログラミング言語であり、構文はC言語や、C言語風に構文が設計されたC++やJavaなどの影響があるが、構文以外についてはヘルスバーグが以前の所属であるBorlandで設計したDelphiからの影響がある

C

C言語は、1972年にAT&Tベル研究所のデニス・リッチーが主体となって開発したプログラミング言語である

Python

Pythonは、汎用のプログラミング言語である

C++

C++は、汎用プログラミング言語の一つである。日本語では略してシープラプラ、シープラなどとも呼ばれる。

ASM

アセンブリ言語とは、コンピュータ、マイクロコントローラ、その他のプログラム可能な機器を動作させるための機械語を人間にわかりやすい形で記述する、代表的な低水準言語である。

Swift

Swiftは、アップルのiOSおよびmacOS､Linuxで利用出来るプログラミング言語。

JavaScript

JavaScriptとは、プログラミング言語のひとつである。Javaと名前が似ているが、全く異なるプログラミング言語である。

Haskell

Haskellは非正格な評価を特徴とする純粋関数型プログラミング言語である。名称は数学者であり論理学者であるハスケル・カリーに由来する。

PHP

PHPは "The PHP Group" によってコミュニティベースで開発されているオープンソースの汎用プログラミング言語およびその公式の処理系であり、特にサーバーサイドで動的なウェブページ作成するための機能を多く備えていることを特徴とする。

MATLAB

MATLABは、アメリカ合衆国のMathWorks社が開発している数値解析ソフトウェアであり、その中で使うプログラミング言語の名称でもある。

ADA

Ada（エイダ）は、強力な言語機能を豊富に持ち、高度な型の体系をもつ、プログラミング言語の一つである。

Otras tecnologías

HTML5

HTMLは、ハイパーテキストを記述するためのマークアップ言語の1つである。

Bootstrap

BootstrapはウェブサイトやWebアプリケーションを作成するフロントエンドWebアプリケーションフレームワークである。

CSS3

CSSとは、HTML や XML の要素をどのように修飾（表示）するかを指示する、W3Cによる仕様の一つ。

JSP

JSPは、HTML内にJavaのコードを埋め込んでおき、Webサーバで動的にWebページを生成してクライアントに返す技術のこと。

JSF

JSFは、JavaベースのWebアプリケーションフレームワークである。Javaの拡張機能セットであるJava EEの仕様の一つに採用されている。

Git

Gitは、プログラムのソースコードなどの変更履歴を記録・追跡するための分散型バージョン管理システムである。

XML

XMLは、基本的な構文規則を共通とすることで、任意の用途向けの言語に拡張することを容易としたことが特徴のマークアップ言語の総称である。

JSON

JSONは軽量なデータ記述言語の1つである。

SQL

SQLは、関係データベース管理システムにおいて、データの操作や定義を行うためのデータベース言語、ドメイン固有言語である。

jQuery

jQueryは、ウェブブラウザ用のJavaScriptコードをより容易に記述できるようにするために設計されたJavaScriptライブラリである。

UML

統一モデリング言語は、主にオブジェクト指向分析や設計のための、記法の統一がはかられたモデリング言語である。

REST

RESTは、ウェブのような分散ハイパーメディアシステムのためのソフトウェアアーキテクチャのスタイルのひとつである。

Sistemas Operativos

Windows

Línea de sistemas operativos producida por Microsoft Corporation.

MacOS

Entorno operativo basado en Unix, desarrollado, comercializado y vendido por Apple Inc.

Linux

Sistema operativo libre tipo Unix; multiplataforma, multi-usuario y multi-tarea.

Android

Sistema operativo basado en el núcleo Linux, diseñado principalmente para dispositivos móviles con pantalla táctil.

iOS

Sistema operativo móvil de la multinacional Apple Inc. Originalmente desarrollado para iPhone, después se ha usado en dispositivos como el iPod touch y el iPad.

Bases de Datos

MySQL

Sistema gestor de bases de datos relacional desarrollado por Oracle Corporation, considerada la base datos open source más popular del mundo.

Microsoft SQL Server

Sistema de manejo de bases de datos del modelo relacional, desarrollado por la empresa Microsoft.

Oracle SQL

Sistema de manejo de datos del modelo relacional de la empresa Oracle Corporation.

MongoDB

Sistema de base de datos NoSQL orientado a documentos, desarrollado bajo código abierto.

SQLite

Sistema de gestión de bases de datos relacional, contenida en una relativamente pequeña biblioteca escrita en C.

Idiomas

Español

Lengua romance procedente del latín hablado. Perteneciente al grupo ibérico, y originaria del Reino de Castilla.

Inglés

Lengua germánica occidental de origen anglosajona de Inglaterra.

Japonés

Idioma de la familia de lenguas japónicas de las lenguas altaicas, hablado por más de 130 millones de personas.
